Documentation/FAQ

Frequently Asked Questions

Common answers for teams configuring and using Dove365 Support.

Setup

Setup & administration

Q

What needs to be configured after installation?

Assign the Support security roles, configure the default queue, confirm the support mailbox automation is enabled, maintain categories and subcategories, and validate the Support dashboards.

Q

What is the default queue used for?

The default queue is the intake queue for new support work. Email triage and available-to-pick dashboard counts rely on it, so the queue GUID must be configured correctly.

Q

Which roles are included?

Dove365 Support uses Support User, Support Account Manager, and Support Admin roles. Assign the least privileged role that lets each person do their work.

Queues

Queues, cases & mailbox triage

Q

Are Queue Items the same as Cases?

No. A Case is the support record. A Queue Item is the queue representation of that record, used for intake, pickup, and work allocation.

Q

Why do Case and Queue Item counts differ?

Dashboards may count active Cases in a user's queue separately from unassigned Queue Items in the default queue. This is expected when work is being triaged or picked up.

Q

Can support emails create Cases automatically?

Yes. The support mailbox triage automation can read incoming support emails, create Cases, and place them into the configured default queue.

Managed services

Managed services & usage

Q

How are support hours tracked?

Support Activities record time against Cases. Managed Service usage can then be aggregated so account managers can review allocation, remaining hours, and usage percentage.

Q

What does the 80 percent threshold mean?

It indicates that a Managed Service is approaching its allocation. Teams should review usage before the allocation is fully consumed.

Q

Who should maintain Managed Services?

Managed Service creation and maintenance is intended for Support Account Managers or Support Admins. Standard Support Users can reference them where permitted.
