Documentation/Environment Variables

Environment Variables Guide

Configure the values Dove365 Support uses for queue intake, mailbox triage, automation routing, and dashboard calculations.

Purpose

Configuration values used by automation

Environment variables let administrators change deployment-specific values without editing flows or code. Dove365 Support uses them to identify the default support queue, support mailbox, and other tenant-specific configuration.

Review these values after solution import, after environment copy, and whenever queue or mailbox ownership changes.
Default Queue GUID

Set the default support intake queue

The default queue is used by mailbox triage, available-to-pick dashboard logic, and queue-based intake reporting.

  1. Open the Queue record that should receive new support work.
  2. Copy the Queue GUID from the record URL.
  3. Set dove365_DefaultQueueGUID to that value.
  4. Run Configure Default Queue manually.
  5. Send a controlled test email and confirm the Case enters the expected queue.

Validation checks

Queue exists and is active
GUID has no extra spaces
Users or teams can access the queue
New Queue Items appear in dashboards
Support Mailbox

Configure mailbox triage

The support mailbox value identifies the mailbox monitored by the intake automation. It should match the mailbox that receives customer support requests.

✓

Confirm the mailbox exists

The mailbox should be active in Microsoft 365 and available to the connection used by the triage flow.

✓

Confirm connections are signed in

Both Outlook and Dataverse connection references need to be authenticated before triage can create Cases.

✓

Test with a controlled email

Use a known test message so you can confirm sender, subject, body, Case creation, and Queue Item placement.

Troubleshooting

Common configuration issues

01

Cases are not entering a queue

Check the default queue GUID, confirm the Configure Default Queue flow has run, and verify the queue is active.

02

Email triage is not creating Cases

Check the support mailbox address, Outlook connection, Dataverse connection, and flow run history.

03

Dashboard counts look wrong

Confirm Queue Items point to active Dove365 Cases and that the default queue value matches the intended intake queue.
