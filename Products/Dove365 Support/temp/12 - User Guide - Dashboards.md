# User Guide - Dashboards

## Purpose

Dove365 Support includes dashboards for personal work management and organisation-wide support operations.

## Dashboard area

Dashboards are accessed inside the Dove365 Support model-driven app.

The main dashboards are:

- Dove365 Support Dashboard
- Dove365 My Support Work Dashboard

## My Support Work Dashboard

Use this dashboard for personal day-to-day work.

It focuses on:

- Cases in your queue.
- Queue Items worked by you.
- Cases available to pick.
- Your upcoming activities.
- Your overdue activities.
- Your Cases requiring attention.

## Dove365 Support Dashboard

Use this dashboard for operational oversight.

It focuses on:

- Open Cases.
- New Cases.
- Critical Cases.
- Triage volume.
- Queue item distribution.
- Queue age.
- Managed Service utilisation.
- Operational tables.

## Using dashboards

Recommended daily routine:

1. Start with My Support Work Dashboard.
2. Review overdue activities.
3. Review Queue Items worked by you.
4. Pick available work if appropriate.
5. Review Cases requiring attention.
6. Use the Support Dashboard for wider team context.

## Troubleshooting dashboard results

If results do not match expectations:

1. Confirm you are in the Dove365 Support app.
2. Confirm security roles.
3. Confirm Queue Items have the correct `Worked By` value.
4. Confirm Cases are active and unresolved.
5. Confirm the default queue GUID is configured.
6. Confirm dashboard filters on the organisation-wide dashboard.
7. Refresh the dashboard.
