Release Notes

Solution-specific release history for Dove365 CRM — Starter and Dove365 Common, including versions 1.1.0 and 1.0.0.

Dove365 CRM — Starter Release Notes

CRM Starter 1.1.0

Market version 1.1.0  ·  Solution package 1.1.0.17  ·  Feature update

Adds the personal My Work dashboard, in-app overview and training content, clearer Lead views, and guidance for optional Edge and Chrome capture extensions.

+

New and changed features

  • Added Dove365 My Work Dashboard with user-owned KPIs for Leads, Opportunities, Cases, Upcoming Tasks, and Overdue Tasks.
  • Added upcoming and overdue activity lists, open activities by type, owned-record tables, mobile layout, loading feedback, diagnostics, and clickable Dynamics rows.
  • Added current-user filtering, Dataverse metadata-based entity set resolution, FetchXML Web API retrieval, and runtime choice labels.
  • Fixed activity date classification by evaluating open owned activities against scheduledend client-side.
  • Added Account Name to all Lead views, using the prospect company name or falling back to the selected existing Account.
  • Added Getting Started overview and CRM Starter training knowledge content.
  • Documented optional Dove365 CRM-Starter Edge and Chrome browser capture extensions.
✓

Post-deployment checks

Install Common before CRM Starter
CRM Starter app opens
My Work KPIs populate for an owner
Dashboard rows open records
Getting Started and training open
Extension users can create permitted records

CRM Starter 1.0.0

Market version 1.0.0  ·  Previous solution 1.0.0.15  ·  Initial release

The initial CRM release introduced the model-driven app, core relationship, pipeline and case records, business process flows, CRM security roles, close/resolve/convert automation, and LinkedIn Badge control.

1

Initial feature set retained

  • Accounts, Contacts, Leads, Opportunities, Cases, Service Offerings, activities, forms, and views.
  • Lead conversion, Opportunity close, and Case resolve processes and Power Automate flows.
  • Dove365 CRM User and CRM Admin security roles.
  • SharePoint documents, Outlook tracking, and LinkedIn Badge support where configured.
Dove365 Common Release Notes

Common 1.1.0

Market version 1.1.0  ·  Solution package 1.1.0.16  ·  Administration update

Expands Dove365 Admin into a broader environment administration hub and adds the Environment Health dashboard.

+

New and changed features

  • Added environment metadata and health KPIs for users, mailboxes, teams, SharePoint locations, Dove365 roles, and installed Dove365 solutions.
  • Added paged Active Users aligned to the Dove365 Users view and Entra-backed user filtering.
  • Fixed organisation version retrieval, readable base currency resolution, and language display fallbacks.
  • Split Dove365 Admin into General Settings and Advanced Settings, with navigation for health, users, teams, mailboxes, service offerings, documents, knowledge, support, security, email, auditing, and system administration.
✓

Post-deployment checks

Dove365 Admin opens for Global Admin
Environment metadata populates
Roles and solutions appear
Users, mailboxes, teams, and locations appear
Advanced Settings links open

Common 1.0.0

Market version 1.0.0  ·  Previous solution 1.0.0.14  ·  Initial release

The initial shared solution provided Service Offerings, shared roles, flow-failure notification, Outlook and Dataverse service-account connections, theme, and app icons.

Version numbers with three places identify the market release. The fourth segment identifies the solution build. For this release, install Dove365 Common 1.1.0.16 before Dove365 CRM — Starter 1.1.0.17.
